The Challenge:
Domain Gap

Models trained on European standards (GTSRB) fail in Southeast Asian environments. This project addresses three specific gaps:

  • Visual Appearance

    Sign designs, fonts, and colors differ significantly between Germany and Vietnam.

  • Semantic Labeling

    Class ID conflicts. E.g., "Stop" is ID 14 in GTSRB but ID 29 in VNTS.

  • Environmental Context

    Different lighting conditions, vegetation, and background clutter.

Performance Analytics

Classification Phase

Accuracy Comparison (Test Set)

Insight: ResNet50 failed (37.9%) because its aggressive downsampling destroyed the information in small 64x64 pixel signs, whereas our Custom CNN (99.7%) preserved spatial details.

Detection Phase

YOLO Model Comparison (mAP 50-95)

Insight: YOLOv8s achieved the highest Recall (95.6%), making it safer for ADAS than the newer YOLOv11n.
